# Fokas

Tired of YouTube Shorts showing up everywhere? Fokas hides all of it — the sidebar tab, the homepage shelf, shorts in search results, the /shorts page. Everything gone.

## What it blocks
- Shorts tab in the sidebar
- Shorts shelf on the homepage
- Shorts in search results
- The entire /shorts page (redirects to home)

## Install
1. Download or clone this repo
2. Go to `chrome://extensions` (or `brave://extensions`)
3. Turn on **Developer mode**
4. Click **Load unpacked** and select the folder
5. Done

## Usage
Click the extension icon in your toolbar and toggle it on. That's it.

## Browser support
Works on any Chromium-based browser — Chrome, Brave, Edge, Opera.
